# Manu Fashions

## Project Overview
Manu Fashions is a fashion e-commerce platform that showcases a variety of clothing products. The aim of the project is to provide a seamless shopping experience for users, allowing them to browse, search, and purchase fashion items easily.

## Features
- User Authentication
- Product Browsing and Searching
- Shopping Cart Functionality
- Secure Payment Integration
- Order Management System

## Tech Stack
- **Frontend:** React, Redux
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Deployment:** Heroku
- **Version Control:** GitHub

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/rockkerorama/manu-fashions.git
   ```
2. Navigate to the project directory:
   ```bash
   cd manu-fashions
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
